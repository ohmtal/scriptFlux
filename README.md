# ScriptFlux - playground for script bindings with BaseFlux


# ChaiScript
Create a Application with a ChaiScript console.

Note: It compiles SDL3, ImGui, BaseFlux and ChaiScript as precompiled headers. Linking takes some time.
Compile and run :
```
cd chaiScript
cmake -S . -B build
cmake --build build
./ChaiFlux
```
First impressions:
    - ➕ Easy to embed.
    - ➕ C-like
    - ➕ Easy to bind
    - ➖ Linking takes ages
